Neverland Vim Theme
====================
All Vim themes suck.

256 colors, CLI only.

![screenshot](http://github.com/trapd00r/neverland-vim-theme/raw/master/screenshot.png)


Installation
============

Copy the theme to $HOME/.vim/colors
